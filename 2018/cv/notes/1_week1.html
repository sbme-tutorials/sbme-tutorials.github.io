Introduction: Python Basics

Variables

Variables in python are very flexible, unlike C++:

  • Interpreted language.
  • No need to declare the variable type.
  • The same variable can be assigned to different types.
x = 5
y = 'Hello SBME'
z = 5
z = "Hello SBME"

If, elif, else

y = 22 

if x < y:
  z = 13 # Local scope

elif x % 2 == 1 and x > y :
  h = 17 # Local scope

else:
  v = 80 # Local scope

Loops

  print( i )

i = 0

Functions

def mean( list ):
  sum = 0
  for element in list:
    sum += element
  return sum / len( sum )

m = mean([1,12,42,1,23,12])
print( m )
